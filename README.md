# Disease Prediction System

This project predicts diseases using Machine Learning techniques and medical datasets.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Seaborn
* Matplotlib

## Machine Learning Algorithms

* Random Forest Classifier
* Logistic Regression

## Features

* Data preprocessing
* Disease prediction
* Model evaluation
* Confusion matrix visualization

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC score

## Dataset

Pima Indians Diabetes Dataset

Dataset Link:
https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database

## Output

Predicts whether a patient is likely to have diabetes or not.

## Tools Used

* Google Colab
* Scikit-learn
* Pandas

## Conclusion

This project demonstrates the use of Machine Learning algorithms for medical disease prediction using patient health data.
